# MutualCircle - الترجمة العربية (Arabic Translation)

## ⚠️ نسخة تجريبية - تحتاج إلى التحقق

**تم إنشاء هذه الترجمات تلقائيًا وتحتاج إلى التحقق من قبل الناطقين الأصليين.**

### الحالة
- **الإصدار**: Alpha 0.1
- **الاكتمال**: 100% (تمت ترجمة جميع النصوص)
- **حالة التحقق**: ❌ لم يتم التحقق من قبل الناطقين الأصليين

### نحتاج مساعدتك!

إذا كنت متحدثًا أصليًا للغة العربية ويمكنك المساعدة في تحسين هذه الترجمات، فسنكون ممتنين للغاية لمساهماتك!

**كيفية المساهمة:**
1. راجع ملفات الترجمة (common.json, auth.json, posts.json, dashboard.json, shifts.json, resources.json)
2. حدد أي ترجمات غير طبيعية أو غير دقيقة أو يمكن تحسينها
3. راجع `CONTRIBUTING_TRANSLATIONS.md` في الجذر الرئيسي للمشروع لإرشادات المساهمة
4. أرسل Pull Request أو أنشئ Issue للإبلاغ عن مشاكل الترجمة

### المشاكل المعروفة
- قد تحتاج بعض المصطلحات التقنية إلى تعديلات للتوطين
- قد تحتاج النبرة ومستوى الرسمية إلى التعديل بناءً على تفضيلات المجتمع
- قد يحتاج المحتوى المرتبط بالثقافة إلى التكيف مع السياق المحلي
- دعم الكتابة من اليمين إلى اليسار (RTL) مُفعّل ولكن قد يحتاج إلى اختبار إضافي

### معلومات الاتصال
- GitHub Issues: للإبلاغ عن مشاكل الترجمة
- البريد الإلكتروني: translations@mutualcircle.org

### We Need Your Help!

If you are a native Arabic speaker and can help improve these translations, we would greatly appreciate your contributions!

**How to Contribute:**
1. Review the translation files (common.json, auth.json, posts.json, dashboard.json, shifts.json, resources.json)
2. Identify any translations that are unnatural, inaccurate, or could be improved
3. See `CONTRIBUTING_TRANSLATIONS.md` in the project root for contribution guidelines
4. Submit a Pull Request or create an Issue to report translation issues

### Known Issues
- Some technical terms may need localization adjustments
- Tone and formality level may need adjustment based on community preferences
- Culture-specific content may need adaptation for local context
- RTL (Right-to-Left) support is enabled but may need additional testing
