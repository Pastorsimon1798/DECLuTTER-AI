### Wir brauchen Ihre Hilfe!

Wenn Sie deutscher Muttersprachler sind und helfen können, diese Übersetzungen zu verbessern, würden wir Ihre Beiträge sehr schätzen!

**Wie Sie beitragen können:**
1. Überprüfen Sie die Übersetzungsdateien (common.json, auth.json, posts.json, dashboard.json, shifts.json, resources.json)
2. Identifizieren Sie Übersetzungen, die unnatürlich, ungenau oder verbesserungswürdig sind
3. Siehe `CONTRIBUTING_TRANSLATIONS.md` im Projektverzeichnis für Beitragsrichtlinien
4. Reichen Sie einen Pull Request ein oder erstellen Sie ein Issue, um Übersetzungsprobleme zu melden

### Bekannte Probleme
- Einige Fachbegriffe benötigen möglicherweise Lokalisierungsanpassungen
- Ton und Formalitätsstufe müssen möglicherweise je nach Community-Präferenzen angepasst werden
- Kulturspezifische Inhalte müssen möglicherweise an den lokalen Kontext angepasst werden
- Unterschiede zwischen deutschem Deutsch, österreichischem Deutsch und Schweizer Deutsch zu berücksichtigen

### Kontakt
- GitHub Issues: Übersetzungsprobleme melden
- E-Mail: translations@mutualcircle.org
