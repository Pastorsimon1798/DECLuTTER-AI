### Kailangan Namin ang Iyong Tulong!

Kung ikaw ay katutubong nagsasalita ng Tagalog at makakatulong na mapabuti ang mga salin na ito, lubos naming papahalagahan ang iyong kontribusyon!

**Paano Mag-contribute:**
1. Suriin ang mga translation file (common.json, auth.json, posts.json, dashboard.json, shifts.json, resources.json)
2. Tukuyin ang anumang salin na hindi natural, hindi tumpak, o maaaring mapabuti
3. Tingnan ang `CONTRIBUTING_TRANSLATIONS.md` sa ugat ng proyekto para sa mga alituntunin sa pag-contribute
4. Magsumite ng Pull Request o lumikha ng Issue upang iulat ang mga problema sa salin

### Kilalang Problema
- Ang ilang terminong teknikal ay maaaring mangailangan ng mga pagsasaayos sa lokalisasyon
- Ang tono at antas ng pormalidad ay maaaring mangailangan ng pagsasaayos batay sa kagustuhan ng komunidad
- Ang nilalamang nauugnay sa kultura ay maaaring mangailangan ng adaptasyon para sa lokal na konteksto
- Pinagsama ang Tagalog at English loanwords na karaniwan sa kasalukuyang paggamit
