### हमें आपकी मदद चाहिए!

यदि आप हिंदी के मूल भाषा बोलने वाले हैं और इन अनुवादों को बेहतर बनाने में मदद कर सकते हैं, तो हम आपके योगदान की बहुत सराहना करेंगे!

**योगदान कैसे करें:**
1. अनुवाद फ़ाइलों की समीक्षा करें (common.json, auth.json, posts.json, dashboard.json, shifts.json, resources.json)
2. किसी भी अनुवाद की पहचान करें जो अप्राकृतिक, गलत, या सुधार योग्य हो
3. योगदान दिशानिर्देशों के लिए परियोजना रूट में `CONTRIBUTING_TRANSLATIONS.md` देखें
4. अनुवाद मुद्दों की रिपोर्ट करने के लिए Pull Request सबमिट करें या Issue बनाएं

### ज्ञात मुद्दे
- कुछ तकनीकी शब्दों को स्थानीयकरण समायोजन की आवश्यकता हो सकती है
- स्वर और औपचारिकता स्तर को समुदाय की प्राथमिकताओं के आधार पर समायोजन की आवश्यकता हो सकती है
- सांस्कृतिक-विशिष्ट सामग्री को स्थानीय संदर्भ के लिए अनुकूलन की आवश्यकता हो सकती है
- देवनागरी लिपि शामिल है लेकिन अतिरिक्त सत्यापन की आवश्यकता हो सकती है

### संपर्क
- GitHub Issues: अनुवाद समस्याओं की रिपोर्ट करें
- ईमेल: translations@mutualcircle.org
