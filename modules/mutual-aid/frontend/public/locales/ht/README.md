**Kijan pou Kontribye:**
1. Revize fichye tradiksyon yo (common.json, auth.json, posts.json, dashboard.json, shifts.json, resources.json)
2. Idantifye nenpòt tradiksyon ki pa natirèl, ki pa egzat, oswa ki ka amelyore
3. Gade `CONTRIBUTING_TRANSLATIONS.md` nan rasin pwojè a pou règleman kontribisyon

- Òtograf kreyòl ka varye - nou itilize estanda ki pi komen
