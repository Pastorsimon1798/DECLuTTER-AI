### ご協力をお願いします！

日本語を母語とする方で、これらの翻訳の改善にご協力いただける場合は、ぜひご貢献ください！

**貢献方法:**
1. 翻訳ファイル（common.json、auth.json、posts.json、dashboard.json、shifts.json、resources.json）を確認してください
2. 不自然、不正確、または改善が必要な翻訳を特定してください
3. プロジェクトルートの `CONTRIBUTING_TRANSLATIONS.md` で貢献ガイドラインをご覧ください
4. プルリクエストを送信するか、翻訳の問題を報告するIssueを作成してください

### 既知の問題
- 一部の技術用語はローカライゼーション調整が必要な場合があります
- コミュニティの好みに応じて、トーンや丁寧さのレベルを調整する必要がある場合があります
- 文化特有のコンテンツは、ローカルコンテキストに適応する必要がある場合があります
- 漢字、ひらがな、カタカナの使い分けについて追加の検証が必要な場合があります
- 敬語の使用レベルと一貫性の確認

### 連絡先
- GitHub Issues: 翻訳の問題を報告
- Email: translations@mutualcircle.org

If you are a native Japanese speaker and can help improve these translations, we would greatly appreciate your contributions!

**How to Contribute:**
1. Review the translation files (common.json, auth.json, posts.json, dashboard.json, shifts.json, resources.json)
2. Identify any translations that are unnatural, inaccurate, or could be improved
3. See `CONTRIBUTING_TRANSLATIONS.md` in the project root for contribution guidelines
4. Submit a Pull Request or create an Issue to report translation issues

### Known Issues
- Some technical terms may need localization adjustments
- Tone and formality level may need adjustment based on community preferences
- Culture-specific content may need adaptation for local context
- Kanji, Hiragana, and Katakana usage may need additional verification
