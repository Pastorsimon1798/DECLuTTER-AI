# Auto-Reply Templates

## Facebook Native Auto-Reply (built-in)

Enable in FB Marketplace listing settings. This handles the #1 most common message automatically: "Is this still available?"

## FB Auto Reply AI Templates

Set these up in the extension for common questions beyond "still available."

### General Availability
```
Yes, still available! Local pickup in Long Beach, CA.
I'm flexible on meetup times — when works for you?
```

### Price Negotiation — Firm
```
Price is firm at $[X]. It's in [condition] condition and retail is $[original price].
Happy to answer any other questions!
```

### Price Negotiation — Flexible
```
I could do $[lower price] if you can pick up today. Otherwise $[current price].
```

### Meetup Scheduling
```
Great! I'm available [days/times]. I usually meet at the [location] parking lot — it's a safe exchange zone. Does that work for you?
```

### For the E-Bike Specifically — Test Rides
```
Yes, you can test ride it! I'll need to hold your car keys (or full payment) during the ride for security. The battery is fully charged and ready to go.
```

### For the E-Bike — Battery/Range Questions
```
Battery is [condition — fully charged / holds full charge]. Range is up to [X] miles on a single charge. I can show you the battery level and charge cycle count when you come see it.
```

### Electronics — Condition Questions
```
[Item] is in [condition] condition. [Specifics: "No scratches on screen," "All buttons work," "Battery health is X%"]. I can power it on and show you everything when we meet up.
```

### "Can you hold it for me?"
```
I can hold it for 24 hours with a $[X] Zelle deposit. After that it goes back up for sale.
```

### "Would you take $[lowball]?"
```
Thanks for the offer, but I'm not quite that low. I could do $[counter — $25-50 below asking] if you can pick up [today/tomorrow].
```

### "Can you ship it?"
```
Sorry, local pickup only — Long Beach, CA. No shipping.
```

### "Is this a scam?"
```
Not a scam! Happy to meet at the Long Beach Police Station safe exchange zone. Cash, Zelle, or Venmo only. You can inspect everything before paying.
```
