# Platform Guide — Long Beach Area

## Platform Comparison

| Platform | Best For | Fees | Listing Limit | Auto-Relist | Buyer Pool |
|----------|----------|------|---------------|-------------|------------|
| **FB Marketplace** | Everything, fastest sales | $0 | Unlimited | Via extension | Largest |
| **OfferUp** | Electronics, bikes | $0 (local) | Unlimited | Built-in | Good |
| **Craigslist** | Big items, e-bikes | $0 | 1 per 48h | No (repost manually) | Medium |
| **Reddit** | Niche items (e-bikes) | $0 | Varies by sub | No | Niche but motivated |
| **FB Groups** | E-bikes, local community | $0 | Varies | No | Targeted |

## Platform-Specific Tips

### Facebook Marketplace
- **Best time to post:** Tuesday-Thursday 10am-12pm
- **Photos:** First photo is your thumbnail — make it count
- **Price:** Set slightly below market to appear in "Deals"
- **Boost:** Not worth it for local items
- **Auto-reply:** Enable native AI auto-reply for "still available?"
- **Relisting:** Use FB Marketplace Relister extension to bump every 3-7 days

### OfferUp
- **App only** — no web posting
- **Pro tip:** List on Wednesday, that's when buyer activity peaks
- **Response rate matters:** Reply within 1 hour for best visibility
- **Trusted seller:** Quick responses and good reviews boost your ranking
- **Bumping:** OfferUp lets you "bump" for free every few days

### Craigslist
- **Title matters most:** Include brand, model, condition, location
- **Photos:** Up to 24 allowed — use them
- **Reposting:** Delete and re-list every 3-7 days to stay near the top
- **No auto-posting tools** — all manual (ToS violation otherwise)
- **Craigslist flagging:** Don't post the same item to multiple categories
- **Best time:** Friday-Sunday morning

### Reddit
- **r/ebikes:** Good for e-bikes, but read subreddit rules first
- **r/BikeLA:** LA-specific bike community
- **r/longbeach:** Local community
- **Post format:** Use imgur for photos, link in post
- **Flair:** Use the correct selling/trade flair
- **Warning:** Reddit can be harsh — follow posting rules exactly

### Facebook Groups
Post to these for the e-bike:
- Los Angeles Electric Bikes
- SoCal eBike Riders
- Long Beach Electric Bike Community
- Electric Bikes for Sale - Southern California

**Group tips:**
- Read group rules before posting (some have specific formats)
- Don't spam — one post per group
- Answer questions quickly
- Bump by commenting "still available" after a few days

## Chrome Extensions Setup Guide

### PopPublic (Autofill)
1. Go to Chrome Web Store → search "PopPublic"
2. Click "Add to Chrome"
3. Open FB Marketplace → "Create new listing"
4. PopPublic sidebar appears — fill in your details once
5. Click "Autofill" on any future listing to populate fields

### FB Marketplace Relister
1. Chrome Web Store → search "FB Marketplace Relister"
2. Add to Chrome
3. Go to your FB Marketplace listings
4. Select listings to auto-bump
5. Set interval (3-7 days recommended)

### FB Auto Reply AI
1. Chrome Web Store → search "FB Auto Reply AI"
2. Add to Chrome
3. Open FB Marketplace → Messages
4. Set up AI response rules and templates
5. Test with a sample message
