**이 번역은 자동으로 생성되었으며 원어민의 검증이 필요합니다.**

### 상태
- **버전**: Alpha 0.1
- **완성도**: 100% (모든 문자열 번역 완료)
- **검증 상태**: ❌ 원어민 검증 미완료

### 여러분의 도움이 필요합니다!

한국어를 모국어로 사용하시며 이 번역 개선에 도움을 주실 수 있다면, 여러분의 기여를 환영합니다!

**기여 방법:**
1. 번역 파일을 검토해 주세요 (common.json, auth.json, posts.json, dashboard.json, shifts.json, resources.json)
2. 부자연스럽거나 부정확하거나 개선이 필요한 번역을 찾아주세요
3. 프로젝트 루트의 `CONTRIBUTING_TRANSLATIONS.md`에서 기여 가이드라인을 확인하세요
4. Pull Request를 제출하거나 번역 문제를 보고하는 Issue를 생성해 주세요

### 알려진 문제
- 일부 기술 용어는 현지화 조정이 필요할 수 있습니다
- 커뮤니티 선호도에 따라 어조와 격식 수준을 조정해야 할 수 있습니다
- 문화 특정 콘텐츠는 현지 상황에 맞게 조정이 필요할 수 있습니다
- 존댓말과 반말의 사용이 적절한지 추가 검토가 필요합니다
- 한국 문화에 맞는 용어 및 표현의 자연스러움 확인

### 연락처
- GitHub Issues: 번역 문제 보고

### We Need Your Help!

If you are a native Korean speaker and can help improve these translations, we would greatly appreciate your contributions!

**How to Contribute:**
1. Review the translation files (common.json, auth.json, posts.json, dashboard.json, shifts.json, resources.json)
2. Identify any translations that are unnatural, inaccurate, or could be improved
3. See `CONTRIBUTING_TRANSLATIONS.md` in the project root for contribution guidelines
4. Submit a Pull Request or create an Issue to report translation issues

### Known Issues
- Some technical terms may need localization adjustments
- Tone and formality level may need adjustment based on community preferences
- Culture-specific content may need adaptation for local context
- Honorific vs. casual language usage needs additional review
- Natural terminology and expressions aligned with Korean culture need verification
