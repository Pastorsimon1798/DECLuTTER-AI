### Нам нужна ваша помощь!

Если вы являетесь носителем русского языка и можете помочь улучшить эти переводы, мы будем очень признательны за ваш вклад!

**Как внести свой вклад:**
1. Просмотрите файлы перевода (common.json, auth.json, posts.json, dashboard.json, shifts.json, resources.json)
2. Определите любые переводы, которые неестественны, неточны или могут быть улучшены
3. См. `CONTRIBUTING_TRANSLATIONS.md` в корне проекта для руководства по внесению вклада
4. Отправьте Pull Request или создайте Issue для сообщения о проблемах с переводом

### Известные проблемы
- Некоторые технические термины могут требовать адаптации для локализации
- Тон и уровень формальности могут требовать корректировки в зависимости от предпочтений сообщества
- Культурно-специфический контент может требовать адаптации для местного контекста
- Кириллица включена, но может требовать дополнительной проверки

### Контакты
- GitHub Issues: Сообщайте о проблемах с переводом
- Email: translations@mutualcircle.org
