# DECLuTTER Unified Product Map

DECLuTTER-AI is the canonical product repo for neurodivergent-friendly home/life support.

Imported modules preserved with git history:

- `modules/mutual-aid/` from `Pastorsimon1798/MutualAidApp`
- `modules/neurodivergent-directory/` from `Pastorsimon1798/neurodivergent-directory`

## Product modules

1. Declutter decisions: photo-based item grouping and keep/donate/trash/relocate/maybe flows.
2. Local trade/donation: turn decluttered items into offers for nearby people.
3. Mutual aid: resource locator, needs/offers board, volunteer coordination, pods/micro-circles.
4. Neurodivergent directory: culturally affirming providers, tools, resources, and verification workflow.

## Integration rule

The imported modules are preserved under `modules/` first. Do not mix code directly into `app/` or `server/` until each feature has an explicit integration spec and tests.
